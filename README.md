# FlexCLI

FlexCLI is the command-line interface for the FlexCoach platform.

## Features
- Context management
- Goal tracking
- Constraint configuration
- Training plan access
- Statistics viewing

## Installation
```bash
brew install f1dot4/flexcoach/flexcli
```
(Installation instructions will be updated once the tap is ready)

## Development
```bash
go build ./cmd/flexcli
```
